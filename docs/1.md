# Setup Guide

## Docker Compose PostgreSQL Setup

The project uses Docker Compose to run a PostgreSQL database. The configuration is defined in `docker-compose.yml`:

```yaml
services:
  postgres:
    image: postgres:17.0
    ports:
      - ${DB_PORT}:5432
    environment:
      - POSTGRES_USER=${DB_USER}
      - POSTGRES_PASSWORD=${DB_PASSWORD}
      - POSTGRES_DB=${DB_NAME}
    volumes:
      - postgres_data:/var/lib/postgresql/data

volumes:
  postgres_data:
```

### Docker Commands

Start the PostgreSQL container:

```bash
docker compose up -d
```

Check running containers:

```bash
docker ps
```

## Prisma Setup

Install Prisma CLI as a dev dependency:

```bash
pnpm add prisma -D
```

Install Prisma Client and dotenv:

```bash
pnpm add dotenv @prisma/client
```

Initialize Prisma:

```bash
pnpm dlx prisma init
```

Push the schema to the database:

```bash
pnpm dlx prisma db push
```
